# TGSM Demo Site

Static landing page for the Triangle Geometric Semantic Model (TGSM) demo.

- Live demo: https://centauriuniverse.shinyapps.io/tgsmv-one/
- Open `index.html` locally or host via GitHub Pages.

## Structure
- `index.html` — main page
- `assets/og-image.png` — social preview image
- `assets/violin.png` — **replace with your provided violin image** (same filename)
- `favicon.svg`, `apple-touch-icon.png`, `site.webmanifest`
- `resources/` — placeholder for docs
